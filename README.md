# Git-Workshop

Git-Workshop is a compact, beginner-friendly learning kit that helps new developers
and students practice everyday Git workflows. It includes a quick-reference cheatsheet
and a set of hands-on exercises designed to teach practical skills such as initializing
repositories, creating branches, merging changes, undoing mistakes, using the stash,
and collaborating with remotes.

Contents
- GIT-CHEATSHEET.md — concise command reference with short examples
- EXERCISES.md — six beginner-friendly exercises with step-by-step instructions

Who should use this
- Beginners learning Git fundamentals
- Instructors preparing short classroom or workshop activities
- Any developer looking for a quick Git refresher

How to use
1. Open a terminal in this folder ([Git-Workshop/README.md](Git-Workshop/README.md)).
2. Read `GIT-CHEATSHEET.md` for command syntax and examples.
3. Complete the exercises in `EXERCISES.md` by following the steps and verifying
	the expected outcomes.
